# C# Learning Exercises

A collection of C# programs written while learning the language basics.

## 📂 Topics Covered

- **Data Types** — Numbers, strings, and type conversion
- **Operators** — Arithmetic and logical operators
- **Arrays** — Array operations and manipulation
- **Control Flow** — Conditions and loops
- **Methods** — Functions and method basics
- **DateTime** — Date and time handling
- **Error Handling** — Try-catch and exception handling
